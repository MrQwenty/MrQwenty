### 🚀 What I'm building

#### 🎙️ [Hitoo](https://hitoo.io) · Real-Time Voice Translation
We're training **AURIS**, a proprietary end-to-end **speech-to-speech model** for real-time multilingual conversation. The pipeline is engineered for sub-300ms end-to-end latency with cross-language voice cloning, running on **mediasoup** for WebRTC SFU.

> 🧠 Training on **50,000 GPU hours** awarded via **EuroHPC AI Factory Fast Lane** on CINECA's Leonardo BOOSTER (A100s)
> 🌍 Selected for **Startup Grind Global 2026** as the only Italian startup among 150 worldwide

#### 🧩 Trim · Modular ERP Framework
Personal engineering project I'm most proud of: modular ERP built around the **Lifecycle Management (LCM)** pattern, using **Go**, **gRPC**, **Redis Streams**, and **Saga choreography** for distributed transactions. Independently deployable services, holds up cleanly under real production load.
